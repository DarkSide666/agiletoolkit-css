Layout

Agile Toolkit CSS allows you to create any desirable page layouts without major changes in a page structure.

Basic layout

Start with this example code and create as many sections as you need with atk-layout-row. Toolkit's layout element covers 100% of available browser space and divides it between layout rows.

<div class="atk-layout">
  <div class="atk-layout-row">
    <div class="atk-layout-cell">Header</div>
  </div>
  <div class="atk-layout-row">
    <div class="atk-layout-cell">Content</div>
  </div>
  <div class="atk-layout-row">
    <div class="atk-layout-cell">Footer</div>
  </div>
</div>

Layout expander

Add a .atk-layout-expand class to expand one of the layout rows. This comes especially handy when you need to create a sticky footer.

<div class="atk-layout">
  <div class="atk-layout-row">
    <div class="atk-layout-cell">Header</div>
  </div>
  <div class="atk-layout-row atk-layout-expand">
    <div class="atk-layout-cell">Content</div>
  </div>
  <div class="atk-layout-row">
    <div class="atk-layout-cell">Sticky Footer</div>
  </div>
</div>

Wrappers

Use .atk-wrapper to keep your content centered and responsive.

<div class="atk-layout">
  <div class="atk-layout-row">
    <div class="atk-layout-cell">
      <div class="atk-wrapper">Header</div>
    </div>
  </div>
  <div class="atk-layout-row atk-layout-expand">
    <div class="atk-layout-cell">
      <div class="atk-wrapper">Content</div>
    </div>
  </div>
  <div class="atk-layout-row">
    <div class="atk-layout-cell">
      <div class="atk-wrapper">Footer</div>
    </div>
  </div>
</div>

Layout columns

Create as many columns as you need with the help of a .atk-layout-column. Same as rows, columns can be expanded with a 'atk-layout-expand' class.

<div class="atk-layout">
  <div class="atk-layout-row">
    <div class="atk-layout-cell">
      <div class="atk-wrapper">Header</div>
    </div>
  </div>
  <div class="atk-layout-row atk-layout-expand">
    <div class="atk-layout-cell atk-layout">
      <div class="atk-layout-column">Left Sidebar</div>
      <div class="atk-layout-column atk-layout-expand">
        <div class="atk-wrapper">Content</div>
      </div>
      <div class="atk-layout-column">Right Sidebar</div>
    </div>
  </div>
  <div class="atk-layout-row">
    <div class="atk-layout-cell">
      <div class="atk-wrapper">Footer</div>
    </div>
  </div>
</div>

Vertical alignment of content

Combine .atk-valign-* classes with layout columns to align content vertically. Works like a charm when you need to center that login box.

<div class="atk-layout">
  <div class="atk-layout-row">
    <div class="atk-layout-cell">Header</div>
  </div>
  <div class="atk-layout-row atk-layout-expand">
    <div class="atk-layout-cell atk-layout">
      <div class="atk-layout-column atk-layout-expand atk-valign-middle">
        Centered Content
      </div>
    </div>
  </div>
  <div class="atk-layout-row">
    <div class="atk-layout-cell">Footer</div>
  </div>
</div>
